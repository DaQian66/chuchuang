网上衣橱系统 - 后端服务已启动

接口基路径: /wardrobe_back

  • 登录注册: POST /wardrobe_back/login?action=register|login|adminLogin
  • 服装接口: GET /wardrobe_back/allClothes?action=details&id=1
  • 购物车: POST /wardrobe_back/cart?action=addToCart (需token)
  • 订单: POST /wardrobe_back/order?action=create (需token)
  • 用户信息: GET /wardrobe_back/user?action=info (需token)
  • 图片上传: POST /wardrobe_back/upload
